# 05 - 数据库设计文档

## 📌 文档说明
本文档详细描述数据库表结构设计、字段定义、索引设计、关系设计,为数据库开发提供完整指导。

---

## 1. 数据库概述

### 1.1 数据库选型
- **主数据库**: PostgreSQL 15+
- **缓存**: Redis 7.x
- **字符集**: UTF-8
- **排序规则**: utf8mb4_general_ci

### 1.2 命名规范
- **表名**: 小写字母+下划线,复数形式,如 `users`, `products`
- **字段名**: 小写字母+下划线,如 `user_id`, `created_at`
- **索引名**: `idx_表名_字段名`,如 `idx_users_phone`
- **外键名**: `fk_表名_字段名`,如 `fk_orders_user_id`

### 1.3 通用字段
所有表都包含以下字段:
- `id`: 主键,自增整数
- `created_at`: 创建时间,TIMESTAMP
- `updated_at`: 更新时间,TIMESTAMP

## 2. 核心表设计

### 2.1 用户表 (users)

**表说明**: 存储普通用户信息

| 字段名 | 类型 | 长度 | 允许空 | 默认值 | 说明 |
|-------|------|------|--------|--------|------|
| id | INT | - | NO | AUTO | 用户ID |
| username | VARCHAR | 50 | NO | - | 用户名 |
| phone | VARCHAR | 11 | NO | - | 手机号 |
| email | VARCHAR | 100 | YES | NULL | 邮箱 |
| password_hash | VARCHAR | 255 | NO | - | 密码哈希 |
| avatar | VARCHAR | 500 | YES | NULL | 头像URL |
| nickname | VARCHAR | 50 | YES | NULL | 昵称 |
| status | ENUM | - | NO | normal | 状态:normal,disabled |
| created_at | TIMESTAMP | - | NO | NOW() | 创建时间 |
| updated_at | TIMESTAMP | - | NO | NOW() | 更新时间 |

**索引**:
- PRIMARY KEY (`id`)
- UNIQUE KEY `uk_username` (`username`)
- UNIQUE KEY `uk_phone` (`phone`)
- UNIQUE KEY `uk_email` (`email`)
- INDEX `idx_status` (`status`)

**SQL**:
```sql
CREATE TABLE users (
    id SERIAL PRIMARY KEY,
    username VARCHAR(50) NOT NULL UNIQUE,
    phone VARCHAR(11) NOT NULL UNIQUE,
    email VARCHAR(100) UNIQUE,
    password_hash VARCHAR(255) NOT NULL,
    avatar VARCHAR(500),
    nickname VARCHAR(50),
    status VARCHAR(20) NOT NULL DEFAULT 'normal',
    created_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_users_status ON users(status);
```

## 3. ER图

```
users (用户)
  ├─ 1:N → addresses (收货地址)
  ├─ 1:N → carts (购物车)
  └─ 1:N → orders (订单)

merchants (商家)
  ├─ 1:N → products (商品)
  └─ 1:N → orders (订单)

categories (分类)
  └─ 1:N → products (商品)

products (商品)
  └─ 1:N → product_skus (SKU)

orders (订单)
  └─ 1:N → order_items (订单商品)
```
