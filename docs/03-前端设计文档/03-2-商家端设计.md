# 03-2 商家端设计文档

## 📌 文档说明
本文档详细描述商家端(B端)的页面设计、功能模块、组件设计、开发顺序,指导商家端的完整开发过程。

---

## 1. 商家端概述

### 1.1 定位
面向入驻商家的管理后台,提供商品管理、订单处理、数据统计等功能。

### 1.2 设计原则
- **效率优先**: 快速完成日常运营任务
- **数据可视化**: 清晰的数据展示和统计
- **操作便捷**: 批量操作、快捷键支持
- **专业性**: 符合B端产品的专业性要求

### 1.3 技术栈
- **框架**: React 18 + TypeScript
- **构建**: Vite
- **UI**: Ant Design (企业级组件库)
- **图标**: Ant Design Icons
- **状态**: Zustand + React Query
- **路由**: React Router 6
- **表单**: Ant Design Form
- **图表**: Apache ECharts

   - 保存草稿(仓库中)
   - 提交审核(等待平台审核)

```typescript
interface ProductForm {
```
