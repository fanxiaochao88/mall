# 03-3 管理端设计文档

## 📌 文档说明
本文档详细描述管理端(Admin)的页面设计、功能模块、组件设计、开发顺序,指导管理端的完整开发过程。

---

## 1. 管理端概述

### 1.1 定位
面向平台管理员的后台管理系统,负责平台运营、商家审核、数据监控等功能。

### 1.2 设计原则
- **权限严格**: 基于角色的权限控制
- **数据全面**: 全局数据监控和统计
- **操作审慎**: 关键操作二次确认
- **日志完整**: 记录所有管理操作

### 1.3 技术栈
- **框架**: React 18 + TypeScript
- **构建**: Vite
- **UI**: Ant Design Pro Components
- **图标**: Ant Design Icons
- **状态**: Zustand + React Query
- **路由**: React Router 6
- **图表**: Apache ECharts
- **表格**: Ant Design Pro Table

  const admin = useAdminStore();

  }
