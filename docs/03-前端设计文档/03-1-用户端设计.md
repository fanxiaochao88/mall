# 03-1 用户端设计文档

## 📌 文档说明
本文档详细描述用户端(C端)的页面设计、功能模块、组件设计、开发顺序,指导用户端的完整开发过程。

---

## 1. 用户端概述

### 1.1 定位
面向普通消费者的购物平台,提供商品浏览、购买、订单管理等功能。

### 1.2 设计原则
- **简洁美观**: 现代化UI,符合主流电商审美
- **易用性**: 操作流程简单,减少用户思考成本
- **响应式**: 支持PC和移动端自适应
- **性能优先**: 快速加载,流畅体验

### 1.3 技术栈
- **框架**: React 18 + TypeScript
- **构建**: Vite
- **UI**: TailwindCSS + Headless UI
- **图标**: Lucide React
- **状态**: Zustand + React Query
- **路由**: React Router 6
- **表单**: React Hook Form + Zod

## 3. 核心页面设计

### 3.1 首页 (/)

#### 页面结构
```
1. 轮播图(Banner)
   - 5张轮播图,自动播放
   - 点击跳转到活动页/商品详情

2. 商品分类导航
   - 一级分类横向展示
   - Hover显示二级分类

3. 推荐商品区
   - 热门推荐
   - 新品上市
   - 限时秒杀(未来)

4. 楼层展示
   - 按分类展示商品(数码、服装、食品等)

```

```tsx
<HomePage>
  <Banner slides={banners} />

  <FloorSection category="数码" products={digitalProducts} />
  <FloorSection category="服装" products={clothingProducts} />
```


#### 页面结构

#### 页面结构
```
┌────────────────────────────────────────┐
│  我的购物车                             │
├─┬──────────────────────────────────────┤
│☑│ 全选                                 │
├─┼────┬─────┬──────┬──────┬──────┬─────┤
│☑│图片│商品 │ 单价 │ 数量 │ 小计 │操作 │
```

#### 核心功能
1. **商品列表**
   - 显示商品图片、名称、规格、价格
   - 修改数量
   - 删除商品

2. **全选/单选**
   - 全选checkbox
   - 单个商品checkbox
   - 计算选中商品总价

3. **失效商品**
   - 下架商品置灰
   - 库存不足提示

4. **结算**
   - 跳转到确认订单页
   - 携带选中的商品ID

#### 状态管理

### 4.2 ProductCard组件

```tsx
interface ProductCardProps {
  product: Product;
  layout?: 'grid' | 'list';
}

<ProductCard product={product}>
  <Image src={product.image} />
  <Title>{product.name}</Title>
  <Price>{product.price}</Price>
  <Sales>{product.sales}人付款</Sales>
  <Actions>
    <AddToCartButton />
  </Actions>

```

### 4.3 AddressSelector组件

```tsx
interface AddressSelectorProps {
  addresses: Address[];
  selectedId?: number;
  onSelect: (id: number) => void;
  onAdd: () => void;
}

<AddressSelector>
  {addresses.map(addr => (
    <AddressCard 
      key={addr.id}
      address={addr}
      selected={addr.id === selectedId}
      onClick={() => onSelect(addr.id)}
    />
  ))}
  <AddButton onClick={onAdd} />
</AddressSelector>
```
