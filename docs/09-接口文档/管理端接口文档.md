# 管理端接口文档

## 📌 文档说明
本文档详细定义管理端(Admin)所有API接口,包括商家审核、商品审核、用户管理、订单监控、数据统计、系统管理等功能。

---

## 1. 基础信息

### 1.1 接口基础URL
```
开发环境(Mock): http://localhost:3002/api/v1
生产环境: http://your-domain.com/api/v1
```

### 1.2 通用请求头
```http
Content-Type: application/json
Authorization: Bearer {admin_token}  # 管理员Token
```

### 1.3 通用响应格式
```typescript
interface Response<T> {
  code: number;
  message: string;
  data: T | null;
}
```

## 2. 管理员认证模块

### 2.1 管理员登录

**接口**: `POST /admins/login`

**权限**: 公开

**请求参数**:
```typescript
interface AdminLoginRequest {
  username: string;
  password: string;
}
```

**请求示例**:
```json
{
  "username": "admin",
  "password": "admin123"
}
```

**响应数据**:
```typescript
interface Admin {
  id: number;
  username: string;
  role: string;          // super_admin, admin, operator
  status: string;        // normal, disabled
  created_at: string;
}

interface AdminLoginResponse {
  access_token: string;
  token_type: string;
  admin: Admin;
}
```

**响应示例**:
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "token_type": "bearer",
    "admin": {
      "id": 1,
      "username": "admin",
      "role": "super_admin",
      "status": "normal",
      "created_at": "2024-02-24T10:00:00Z"
    }
  }
}
```

**业务规则**:
- 只有`normal`状态的管理员可以登录
- `disabled`状态提示"账号已被禁用"
- 密码错误5次锁定账号30分钟
