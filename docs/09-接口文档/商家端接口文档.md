# 商家端接口文档

## 📌 文档说明
本文档详细定义商家端(B端)所有API接口,包括商家入驻、商品管理、订单管理、数据统计等功能的接口定义。

---

## 1. 基础信息

### 1.1 接口基础URL
```
开发环境(Mock): http://localhost:3001/api/v1
生产环境: http://your-domain.com/api/v1
```

### 1.2 通用请求头
```http
Content-Type: application/json
Authorization: Bearer {merchant_token}  # 商家Token
```

### 1.3 通用响应格式
```typescript
interface Response<T> {
  code: number;
  message: string;
  data: T | null;
}
```

## 2. 商家认证模块

### 2.1 商家注册(入驻申请)

**接口**: `POST /merchants/register`

**权限**: 公开

**请求参数**:
```typescript
interface MerchantRegisterRequest {
  username: string;              // 登录账号, 3-50字符
  password: string;              // 密码, 6-20字符
  shop_name: string;             // 店铺名称
  contact_name: string;          // 联系人
  contact_phone: string;         // 联系电话
  business_license: string;      // 营业执照URL
  id_card_front: string;         // 身份证正面URL
  id_card_back: string;          // 身份证反面URL
  shop_description?: string;     // 店铺简介
}
```

**请求示例**:
```json
{
  "username": "apple_store",
  "password": "123456",
  "shop_name": "Apple官方旗舰店",
  "contact_name": "张三",
  "contact_phone": "13800138000",
  "business_license": "https://example.com/license.jpg",
  "id_card_front": "https://example.com/id-front.jpg",
  "id_card_back": "https://example.com/id-back.jpg",
  "shop_description": "Apple官方授权店铺"
}
```

**响应数据**:
```typescript
interface Merchant {
  id: number;
  username: string;
  shop_name: string;
  shop_logo?: string;
  shop_description?: string;
  contact_name: string;
  contact_phone: string;
  business_license: string;
  id_card_front: string;
  id_card_back: string;
  status: string;              // pending, approved, rejected, disabled
  reject_reason?: string;
  created_at: string;
}
```

**响应示例**:
```json
{
  "code": 201,
  "message": "入驻申请已提交,请等待审核",
  "data": {
    "id": 1,
    "username": "apple_store",
    "shop_name": "Apple官方旗舰店",
    "shop_logo": null,
    "shop_description": "Apple官方授权店铺",
    "contact_name": "张三",
    "contact_phone": "13800138000",
    "business_license": "https://example.com/license.jpg",
    "id_card_front": "https://example.com/id-front.jpg",
    "id_card_back": "https://example.com/id-back.jpg",
    "status": "pending",
    "reject_reason": null,
    "created_at": "2024-02-24T10:00:00Z"
  }
}
```

**业务规则**:
- 用户名唯一
- 注册后状态为`pending`,等待管理员审核
- 审核通过后才能登录

### 2.2 商家登录

**接口**: `POST /merchants/login`

**权限**: 公开

**请求参数**:
```typescript
interface MerchantLoginRequest {
  username: string;
  password: string;
}
```

**请求示例**:
```json
{
  "username": "apple_store",
  "password": "123456"
}
```

**响应数据**:
```typescript
interface MerchantLoginResponse {
  access_token: string;
  token_type: string;
  merchant: Merchant;
}
```

**响应示例**:
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "access_token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "token_type": "bearer",
    "merchant": {
      "id": 1,
      "username": "apple_store",
      "shop_name": "Apple官方旗舰店",
      "shop_logo": "https://example.com/logo.jpg",
      "shop_description": "Apple官方授权店铺",
      "contact_name": "张三",
      "contact_phone": "13800138000",
      "status": "approved",
      "created_at": "2024-02-24T10:00:00Z"
    }
  }
}
```

**业务规则**:
- 只有`approved`状态的商家可以登录
- `pending`状态提示"审核中"
- `rejected`状态提示"审核未通过"
- `disabled`状态提示"账号已被禁用"

### 3.3 发布商品

**接口**: `POST /merchants/products`

**权限**: 需要登录

**请求参数**:
```typescript
interface CreateProductRequest {
  category_id: number;
  name: string;                // 2-200字符
  description: string;         // 富文本详情
  images: string[];            // 1-5张图片
  video?: string;
  skus: SKUInput[];            // 至少1个SKU
}

interface SKUInput {
  sku_name?: string;
  price: number;               // 大于0
  stock: number;               // 大于等于0
  image?: string;
  attributes?: Record<string, string>;
}
```

**请求示例**:
```json
{
  "category_id": 1,
  "name": "iPhone 15 Pro Max",
  "description": "<p>商品详情...</p>",
  "images": [
    "https://example.com/iphone-1.jpg",
    "https://example.com/iphone-2.jpg"
  ],
  "video": null,
  "skus": [
    {
      "sku_name": "深空黑-128G",
      "price": 9999,
      "stock": 100,
      "image": "https://example.com/iphone-black.jpg",
      "attributes": {
        "颜色": "深空黑",
        "容量": "128G"
      }
    },
    {
      "sku_name": "深空黑-256G",
      "price": 10999,
      "stock": 50,
      "image": "https://example.com/iphone-black.jpg",
      "attributes": {
        "颜色": "深空黑",
        "容量": "256G"
      }
    }
  ]
}
```

**响应示例**:
```json
{
  "code": 201,
  "message": "商品发布成功,等待审核",
  "data": {
    "id": 1,
    "merchant_id": 1,
    "category_id": 1,
    "name": "iPhone 15 Pro Max",
    "status": "pending",
    "created_at": "2024-02-24T10:00:00Z"
  }
}
```

**业务规则**:
- 发布后状态为`pending`,等待管理员审核
- 审核通过后状态变为`approved`,自动上架
- 审核拒绝后状态变为`rejected`

**请求示例**:
```json

}
```

**响应示例**:
```json
{

  "message": "商品已下架",
  "data": {
    "id": 1,
    "status": "offline"
  }
}
```

**业务规则**:
- 只有`approved`状态的商品可以下架
