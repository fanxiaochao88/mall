# 商品详情页设计文档

## 一、页面概述

商品详情页是用户从商品列表点击进入的核心页面，展示商品的完整信息，并提供加入购物车和立即购买功能。

**路由**: `/products/:id`

**入口**:
- 首页推荐模块的商品卡片
- 首页楼层模块的商品卡片
- 商品列表页的商品卡片

## 四、核心功能设计

### 4.1 SKU 选择逻辑

这是商品详情页最复杂的部分，需要处理多规格商品的选择。

#### 4.1.1 属性提取
从 `skus` 数组中提取所有可能的属性组合：

```typescript
// 示例数据
skus: [
  { id: 1, attributes: { color: "红色", size: "S" }, price: 99, stock: 10 },
  { id: 2, attributes: { color: "红色", size: "M" }, price: 99, stock: 5 },
  { id: 3, attributes: { color: "蓝色", size: "S" }, price: 109, stock: 0 },
  { id: 4, attributes: { color: "蓝色", size: "M" }, price: 109, stock: 8 },
]

// 提取后的属性结构
attributes: {
  color: ["红色", "蓝色"],
  size: ["S", "M"]
}
```

#### 4.1.2 SKU 匹配算法
用户选择属性后，需要找到对应的 SKU：

```typescript
/**
 * 根据选中的属性组合查找对应的 SKU
 * @param skus - SKU列表
 * @param selectedAttributes - 选中的属性 {color: "红色", size: "M"}
 * @returns 匹配的SKU或null
 */
function findMatchingSKU(
  skus: ProductSKU[],
  selectedAttributes: Record<string, string>
): ProductSKU | null {
  return skus.find(sku => {
    if (!sku.attributes) return false

    // 检查所有属性是否完全匹配
    return Object.keys(selectedAttributes).every(
      key => sku.attributes![key] === selectedAttributes[key]
    )
  }) || null
}
```

#### 4.1.3 禁用状态判断
某些属性组合可能无库存，需要禁用：

```typescript
/**
 * 判断某个属性值是否可选
 * @param attrKey - 属性键（如 "color"）
 * @param attrValue - 属性值（如 "红色"）
 * @param selectedAttributes - 当前已选属性
 * @param skus - SKU列表
 * @returns 是否可选
 */
function isAttributeAvailable(
  attrKey: string,
  attrValue: string,
  selectedAttributes: Record<string, string>,
  skus: ProductSKU[]
): boolean {
  // 构建临时属性组合
  const tempAttrs = { ...selectedAttributes, [attrKey]: attrValue }

  // 查找是否有匹配的SKU且有库存
  return skus.some(sku => {
    if (!sku.attributes || sku.stock <= 0) return false

    // 检查是否匹配（只检查已选择的属性）
    return Object.keys(tempAttrs).every(
      key => sku.attributes![key] === tempAttrs[key]
    )
  })
}
```

### 4.2 价格显示逻辑

```typescript
/**
 * 获取显示价格
 * - 如果选中了SKU，显示该SKU的价格
 * - 如果未选中，显示价格区间
 */
function getDisplayPrice(
  selectedSku: ProductSKU | null,
  skus: ProductSKU[]
): string {
  if (selectedSku) {
    return `¥${selectedSku.price.toFixed(2)}`
  }

  const prices = skus.map(sku => sku.price)
  const minPrice = Math.min(...prices)
  const maxPrice = Math.max(...prices)

  if (minPrice === maxPrice) {
    return `¥${minPrice.toFixed(2)}`
  }

  return `¥${minPrice.toFixed(2)} - ¥${maxPrice.toFixed(2)}`
}
```

### 4.3 库存显示逻辑

```typescript
/**
 * 获取库存信息
 */
function getStockInfo(
  selectedSku: ProductSKU | null,
  skus: ProductSKU[]
): { stock: number; available: boolean } {
  if (selectedSku) {
    return {
      stock: selectedSku.stock,
      available: selectedSku.stock > 0
    }
  }

  // 未选中SKU时，显示总库存
  const totalStock = skus.reduce((sum, sku) => sum + sku.stock, 0)
  return {
    stock: totalStock,
    available: totalStock > 0
  }
}
```

```typescript
 * 数量增加
 */
function handleIncrease(
  currentQuantity: number,
  maxStock: number
): number {
  if (currentQuantity >= maxStock) {
    message.warning('已达库存上限')
    return currentQuantity
  }
  return currentQuantity + 1
}

/**
 * 数量减少
 */
function handleDecrease(currentQuantity: number): number {
  if (currentQuantity <= 1) {
    return 1
  }
  return currentQuantity - 1
}

/**
 * 手动输入数量
 */
function handleQuantityChange(
  value: string,
  maxStock: number
): number {
  const num = parseInt(value, 10)

  if (isNaN(num) || num < 1) {
    return 1
  }

  if (num > maxStock) {
    message.warning('超过库存数量')
    return maxStock
  }

  return num
}
```

### 4.5 加入购物车

```typescript
/**
 * 加入购物车
 */
async function handleAddToCart(
  selectedSku: ProductSKU | null,
  quantity: number
) {
  // 1. 校验是否选择了SKU
  if (!selectedSku) {
    message.warning('请选择商品规格')
    return
  }

  // 2. 校验库存
  if (selectedSku.stock < quantity) {
    message.error('库存不足')
    return
  }

  // 3. 调用API
  try {
    await cartApi.addCart({
      sku_id: selectedSku.id,
      quantity: quantity
    })
    message.success('已加入购物车')
  } catch (error) {
    message.error('加入购物车失败')
  }
}
```

## 六、Hook 封装

### 6.1 useProductDetail.ts
```typescript
// 文件位置: src/hooks/serverData/useProductDetail.ts

import { useQuery } from '@tanstack/react-query'
import { productApi } from '@/api/product'

/**
 * @description 商品详情数据获取 hook
 */
export function useProductDetail(id: number) {
  const { data, isLoading, error } = useQuery({
    queryKey: ['product', id],
    queryFn: () => productApi.getProductDetail(id),
    enabled: !!id && id > 0,  // 只有id有效时才请求
  })

  return {
    product: data,
    isLoading,
    error,
  }
}
```

### 6.2 useSKUSelector.ts（可选）
如果 SKU 逻辑复杂，可以封装成独立 hook：

```typescript
// 文件位置: src/hooks/useSKUSelector.ts

import { useState, useMemo } from 'react'
import type { ProductSKU } from '@/types'

export function useSKUSelector(skus: ProductSKU[]) {
  const [selectedAttributes, setSelectedAttributes] = useState<Record<string, string>>({})

  // 提取所有属性
  const attributes = useMemo(() => {
    const attrs: Record<string, Set<string>> = {}

    skus.forEach(sku => {
      if (sku.attributes) {
        Object.entries(sku.attributes).forEach(([key, value]) => {
          if (!attrs[key]) attrs[key] = new Set()
          attrs[key].add(value)
        })
      }
    })

    // 转换为数组
    return Object.entries(attrs).reduce((acc, [key, values]) => {
      acc[key] = Array.from(values)
      return acc
    }, {} as Record<string, string[]>)
  }, [skus])

  // 查找匹配的SKU
  const selectedSku = useMemo(() => {
    return skus.find(sku => {
      if (!sku.attributes) return false
      return Object.keys(selectedAttributes).every(
        key => sku.attributes![key] === selectedAttributes[key]
      )
    }) || null
  }, [skus, selectedAttributes])

  // 判断属性是否可选
  const isAttributeAvailable = (key: string, value: string) => {
    const tempAttrs = { ...selectedAttributes, [key]: value }
    return skus.some(sku => {
      if (!sku.attributes || sku.stock <= 0) return false
      return Object.keys(tempAttrs).every(
        k => sku.attributes![k] === tempAttrs[k]
      )
    })
  }

  // 选择属性
  const selectAttribute = (key: string, value: string) => {
    setSelectedAttributes(prev => ({ ...prev, [key]: value }))
  }

  return {
    attributes,
    selectedAttributes,
    selectedSku,
    isAttributeAvailable,
    selectAttribute,
  }
}
```

## 八、注意事项

### 8.1 边界情况处理
1. **单 SKU 商品**: 只有一个 SKU，不需要选择，直接显示价格和库存
2. **无库存商品**: 所有 SKU 库存为 0，禁用购买按钮，显示"暂时缺货"
3. **商品下架**: status 为 'off_sale'，显示"商品已下架"
4. **无图片**: 使用默认占位图
5. **无视频**: 不显示视频播放器

### 8.2 用户体验优化
1. **默认选中**: 如果只有一个 SKU，自动选中
2. **智能推荐**: 优先选中库存最多的 SKU
3. **操作反馈**: 所有操作都要有 Toast 提示
4. **加载状态**: 按钮点击后显示 loading
5. **图片预加载**: 提前加载缩略图

### 8.3 性能优化
1. **图片懒加载**: 详情图片使用懒加载
2. **useMemo**: SKU 匹配逻辑使用 useMemo 缓存
3. **防抖**: 数量输入框使用防抖
4. **路由懒加载**: 页面使用 lazy 加载

### 8.4 移动端适配
1. **响应式布局**: 使用 Tailwind 的响应式类
2. **图片轮播**: 移动端使用滑动切换
3. **按钮大小**: 移动端按钮更大，方便点击
4. **字体大小**: 移动端适当调整字体
