## 🔧 配置说明

### API代理配置
开发环境下,所有`/api`请求会被代理到`http://localhost:3000`

### 环境变量
- 开发环境: `.env.development`
- 生产环境: `.env.production`

### 路径别名
使用`@/`作为`src/`的别名,例如:
```typescript
import { userApi } from '@/api'
import { User } from '@/types'
```
